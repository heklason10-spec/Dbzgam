[Game Title]
# Dragon Ball Z-inspired Fighting Game

[Description]
This project aims to create a fighting game inspired by Dragon Ball Z, using the Godot Engine.

[Goals]
- Engage players with fast-paced combat mechanics.
- Include well-known characters and abilities.
- Provide an immersive UI experience.

[Getting Started]
1. Clone the repository.
2. Open the project in Godot Engine.
3. Start developing scenes, scripts, and assets.
